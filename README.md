# Simulasi Aktivitas Perkenalan AI: Emotion Recognition

Proyek ini dibuat khusus untuk keperluan simulasi dan edukasi dalam memahami dasar-dasar Kecerdasan Buatan (AI), khususnya dalam mendeteksi ekspresi emosi (_Emotion Recognition_).

Melalui proyek ini, peserta dapat melihat langsung bagaimana sebuah model AI dilatih, mulai dari memproses data mentah hingga melakukan prediksi.

---

## Struktur Proyek

Di dalam repositori ini, terdapat dua komponen utama yang digunakan untuk simulasi:

1. **`dev_model.ipynb`**
   - **Deskripsi:** Berkas _Jupyter Notebook_ yang berisi seluruh kode sumber (_source code_) untuk proses pengembangan model AI.
2. **`sample_dataset/`**
   - **Deskripsi:** Direktori yang berisi sampel data gambar yang akan digunakan sebagai bahan latihan model.
